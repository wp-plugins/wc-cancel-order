Add Cancel order request button at my account page.2. Customer can send order cancellation request to the store manager or admin.3. Store manager or admin can approve or deny the order cancellation request.== Installation ==1. Simply install and activate the plugin.2. After installation the order cancel request button will appear at woocommerce my account page.3. WC Cancel Order Admin Dashboard >>> woocommerce >> Cancel Requests. == Frequently Asked Questions === having issue or need help with WC Cancel Order plugin ? =Just write at plugin support forum.== Screenshots ==1.
